# fastapi-ml-api-project
This project demonstrates end-to-end backend API development using FastAPI, including CRUD operations, request validation, Machine Learning model integration, and deployment.  The goal of this project is to build production-ready APIs while understanding backend architecture, data validation, and ML model serving.


Project Highlights

Built RESTful APIs using FastAPI

Implemented full CRUD operations using a local JSON database

Applied request & response validation using Pydantic

Structured project for scalable backend development

Integrated Machine Learning model for Car Price Prediction

Connected backend with Streamlit frontend

Deployed FastAPI backend on Render

Deployed Streamlit UI on Streamlit Cloud
